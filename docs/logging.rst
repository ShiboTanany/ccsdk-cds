Logging
-------
.. toctree::
   :maxdepth: 1

CCSDK uses slf4j to log messages to the standard OpenDaylight karaf.log
log file.

Where to Access Information
---------------------------
Logs are found within the SDNC docker container, in the directory
/opt/opendaylight/current/data/logs.
